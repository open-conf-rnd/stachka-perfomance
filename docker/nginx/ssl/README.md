# SSL для локальной разработки

Папка монтируется в nginx. Сертификаты должны быть выпущены **для домена из `.env`** (переменная `DOMAIN`), чтобы Telegram и браузер принимали HTTPS.

Создать сертификаты (домен берётся из `.env` или из аргумента):

```bash
./scripts/gen-ssl.sh
# или для конкретного домена:
./scripts/gen-ssl.sh your-subdomain.ngrok.io
```

В папке появятся `cert.pem` и `key.pem`.
