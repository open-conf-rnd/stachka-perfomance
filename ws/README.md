# WS — живая статистика опросов

WebSocket-сервер для рассылки статистики голосований (плагин Figma, Mini App). Контракт сообщений см. в корневом README или в описании контракта.

- **Порт:** `STATS_WS_PORT` (по умолчанию 3001)
- **Сообщения:** `subscribe` (подписка на опрос), `vote` (голос), рассылка `stats`

## Запуск

```bash
# из корня репозитория
npm run dev:ws

# или из папки ws
npm run dev
```

Подключение: `ws://localhost:3001` или через nginx `wss://<DOMAIN>/ws`.
